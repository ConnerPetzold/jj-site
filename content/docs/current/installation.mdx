<Steps>
  <Step>
     ### Install

     Install on MacOS, Linux and Windows (WSL).

     ```shell
     curl https://jj.sh/install -fsS | bash
     ```

  </Step>
  <Step>
    ### Configure user

     Set your name and email so commits are made in your name.

     ```shell
     jj config set --user user.name "Martin von Zweigbergk"
     jj config set --user user.email "martinvonz@google.com"
     ```

  </Step>
  <Step>
    ### Command-line completion

    Optionally, set up command-line completion for your shell.

    <Tabs items={['Bash', 'Zsh', 'Fish', 'Nushell', 'Xonsh', 'PowerShell']}>
      <Tab value="Bash">
        ```shell
        source <(jj util completion bash)
        ```
      </Tab>
      <Tab value="Zsh">
        ```shell
        autoload -U compinit
        compinit
        source <(jj util completion zsh)
        ```
      </Tab>
      <Tab value="Fish">
        <Callout>
          No configuration is required with fish >= 4.0.2 which loads dynamic completions by default.
        </Callout>
        ```shell
        jj util completion fish | source
        ```
      </Tab>
      <Tab value="Nushell">
        ```shell
        jj util completion nushell | save completions-jj.nu
        use completions-jj.nu * # Or `source completions-jj.nu`
        ```
      </Tab>
      <Tab value="Xonsh">
        ```shell
        source-bash $(jj util completion)
        ```
      </Tab>
      <Tab value="PowerShell">
        Insert this line in your `$PROFILE` file (usually `$HOME\Documents\PowerShell\Microsoft.PowerShell_profile.ps1`):

        ```powershell
        Invoke-Expression (& { (jj util completion power-shell | Out-String) })
        ```
      </Tab>
    </Tabs>

  </Step>
</Steps>
